## 🧱 Field summary

- `domain`, Fully qualified domain name.
- `status`, Current acquisition state for the domain in the public extract.
- `purchase_price`, Visible purchase price when available.
- `renewal_price`, Visible renewal price when available.
- `attractiveness`, Public composite naming band used as a decision-support signal.
- `demand`, Public buyer-pressure band when available.
- `length`, Character count without the TLD.
- `registrar`, Registrar name when known.
- `created_at`, Creation timestamp when known.
- `expires_at`, Expiry timestamp when known.

## ⚠️ Methodology and caveats

This set covers 8,958 one-word .studio domain names, spanning everyday words and short compound terms such as coffeeberry.studio, jewels.studio, and herbaltea.studio. The .studio extension is widely used by design studios, creative agencies, and independent brands looking for a clear, descriptive web address. Median asking price across the selection sits near $43, making most names accessible for early-stage projects. When comparing these domains, prioritize spelling simplicity, brand relevance, and renewal cost over length alone, since a memorable one-word name typically outperforms a longer, more complex alternative.
